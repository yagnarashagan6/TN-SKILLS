JavaScript Event Listeners

What are Event Listeners?
Event listeners allow you to execute code when specific events happen (like clicks, keypresses, mouse movements, etc.). They make web pages interactive!

Basic Syntax

// Select element
const button = document.getElementById('myButton');

// Add event listener
button.addEventListener('click', function() {
  console.log('Button clicked!');
});

// With arrow function
button.addEventListener('click', () => {
  console.log('Button clicked!');
});

Interactive Demos

1. Click Event

button.addEventListener('click', () => {
  console.log('Clicked!');
});
Click the button above

2. Double Click Event

button.addEventListener('dblclick', () => {
  console.log('Double clicked!');
});
Double click the button

3. Mouse Events (hover)

box.addEventListener('mouseover', () => {
  // Mouse enters
});
box.addEventListener('mouseout', () => {
  // Mouse leaves
});
Hover Me!
Hover over the box

4. Input Event

input.addEventListener('input', (e) => {
  console.log(e.target.value);
});
Start typing...

5. Keypress Event

input.addEventListener('keydown', (e) => {
  console.log('Key pressed:', e.key);
});
Press a key in the input

6. Change Event (Select)

select.addEventListener('change', (e) => {
  console.log(e.target.value);
});
Select a color

7. Multiple Events on One Element

button.addEventListener('click', () => {...});
button.addEventListener('mouseover', () => {...});
button.addEventListener('mouseout', () => {...});

Event Object

element.addEventListener('click', (event) => {
  console.log(event.target); // The clicked element
  console.log(event.type); // Event type (e.g., 'click')
  console.log(event.clientX); // Mouse X position
  console.log(event.clientY); // Mouse Y position
});
Click to see event details

Removing Event Listeners

// Named function (required for removal)
function handleClick() {
  console.log('Clicked!');
}

// Add listener
button.addEventListener('click', handleClick);

// Remove listener
button.removeEventListener('click', handleClick);
